# 🚀 Blink – Realtime Chat App

**Blink** is a minimal yet elegant real-time chat application built entirely with HTML, CSS, and Vanilla JavaScript — powered by Firebase Authentication and Firestore. 
It features an engaging, glassmorphic UI with real-time message syncing, user status, and smooth animations, optimized for both desktop and mobile browsers.

---

## ✨ Features

- 🔐 Anonymous Firebase Authentication
- 📡 Real-time Firestore Messaging
- 🎨 Modern Glassmorphism UI (No Frameworks)
- 🧠 Duplicate Username Prevention
- ♻️ Auto-refresh logic for first-time load
- ✅ Message limit (stores last 50 messages)
- 🌐 Fully responsive and mobile-friendly
- ⚡ Connection Status Indicator (Online/Offline)

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS (custom animations & styles), JavaScript (Vanilla)
- **Backend-as-a-Service:** Firebase (Auth + Firestore)
- **Hosting (Optional):** Firebase Hosting


---

## ⚠️ Important Note

**First-time users:** Please refresh the page once after the initial load to ensure Firebase initializes correctly. 
This is due to `sessionStorage` being used to control auto-reload logic.
